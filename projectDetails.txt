1. Project Concept
The External Brain is a hybrid productivity system designed to solve "mental load" failure. It bridges the gap between passive planning (writing tasks down) and active execution (actually doing them).
Unlike a standard To-Do list that sits quietly in an app, this project acts as a persistent personal assistant that uses "Smart Nagging" logic to ensure consistency in fitness, diet, and work.

3. Detailed Project Workflow
Phase 1: The Planning (User $\rightarrow$ Next.js)
Input: You open the dashboard and add a task (e.g., "Buy Chicken") or a daily habit (e.g., "10k Steps").
Sync: Next.js saves this to DynamoDB.
Schedule: An AWS Lambda is triggered to create an EventBridge "One-Time" Schedule for the reminder time.
Phase 2: The Active Nag (AWS $\rightarrow$ User)
